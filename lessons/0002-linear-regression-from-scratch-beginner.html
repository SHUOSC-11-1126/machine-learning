Lesson 0002: Linear Regression From Scratch

第 2 课：从零理解老师的第一部分线性回归代码

Goal in English: Be able to explain and fill in the teacher's MyOwnLinearRegression.fit code.

中文目标：能解释并填出老师的 MyOwnLinearRegression.fit 代码。

1 What problem are we solving?

English: Linear regression predicts a continuous number, such as salary, house price, or score.

中文：线性回归用来预测一个连续数值，比如工资、房价、分数。

English: The data has inputs and true answers.

中文：数据里有输入，也有真实答案。

SymbolEnglish meaning中文
XInput features.输入特征。
yTrue target values.真实答案。
y_predictedModel predictions.模型预测值。

2 What are weights and bias?

English: A weight tells the model how important one feature is. The bias is a constant added to every prediction.

中文：weight 表示某个特征有多重要。bias 是加到每个预测值上的常数。

prediction = feature_1 * weight_1
           + feature_2 * weight_2
           + ...
           + bias

English: If there are n_features features, we need n_features weights.

中文：如果有 n_features 个特征，就需要 n_features 个权重。

n_samples, n_features = X.shape
self.weights = np.zeros(n_features)
self.bias = 0

3 How do we predict?

English: The teacher's prediction line is:

中文：老师的预测代码是：

y_predicted = np.dot(X, self.weights) + self.bias

English: np.dot(X, self.weights) computes one weighted sum per sample.

中文：np.dot(X, self.weights) 会给每个样本算一个加权和。

X = [[1, 2],
     [3, 4]]
weights = [10, 100]
bias = 5

first prediction  = 1*10 + 2*100 + 5 = 215
second prediction = 3*10 + 4*100 + 5 = 435

English: Do not use X * self.weights as the prediction line. It only multiplies element by element and does not sum the features.

中文：不要把 X * self.weights 当作预测代码。它只是逐元素相乘，不会把特征加起来。

4 How do we know the prediction is bad?

English: We compare the prediction with the true answer. The difference is the error.

中文：我们把预测值和真实答案比较，差值就是误差。

error = y_predicted - y

English: MSE loss means average squared error.

中文：MSE loss 是误差平方的平均值。

math:   L = (1/n) * sum((y_hat - y)^2)
Python: loss = np.mean((y_predicted - y) ** 2)

5 What are gradients?

English: A gradient tells us how the loss changes when we change a parameter. In this code, dw is the gradient for weights, and db is the gradient for bias.

中文：梯度告诉我们：当参数变化时，loss 会怎么变。在这段代码里，dw 是 weights 的梯度，db 是 bias 的梯度。

dw = (2 / n_samples) * np.dot(X.T, (y_predicted - y))
db = (2 / n_samples) * np.sum(y_predicted - y)

English: X.T is used so that dw has one gradient value for each feature. Therefore, dw.shape matches self.weights.shape.

中文：这里用 X.T，是为了让 dw 对每个 feature 都有一个梯度值。所以 dw.shape 和 self.weights.shape 一样。

6 How do we improve the model?

English: Gradient descent updates parameters by moving against the gradient.

中文：梯度下降通过“朝梯度相反方向走”来更新参数。

self.weights = self.weights - self.lr * dw
self.bias = self.bias - self.lr * db

English: The minus sign is important. The gradient points toward increasing loss, so we subtract it to reduce the loss.

中文：减号非常重要。梯度指向 loss 增大的方向，所以我们要减去它来降低 loss。

7 The whole fit method

def fit(self, X, y):
    n_samples, n_features = X.shape

    self.weights = np.zeros(n_features)
    self.bias = 0

    for _ in range(self.n_iters):
        y_predicted = np.dot(X, self.weights) + self.bias

        dw = (2 / n_samples) * np.dot(X.T, (y_predicted - y))
        db = (2 / n_samples) * np.sum(y_predicted - y)

        self.weights = self.weights - self.lr * dw
        self.bias = self.bias - self.lr * db

8 Exam question and answer

Question

English: Explain the main steps of the fit method in linear regression from scratch.

中文：解释从零实现线性回归时 fit 方法的主要步骤。

Answer

English: The fit method first reads n_samples and n_features from X.shape. It initializes one weight for each feature and sets the bias to zero. In each iteration, it predicts with np.dot(X, weights) + bias, computes the MSE gradients dw and db, and updates the parameters by subtracting the learning rate times the gradients.

中文：fit 方法先从 X.shape 读出样本数和特征数。它为每个特征初始化一个权重，并把 bias 设为 0。每次迭代中，它用 np.dot(X, weights) + bias 进行预测，计算 MSE 对应的梯度 dw 和 db，然后用“参数减去 learning rate 乘以梯度”的方式更新参数。
