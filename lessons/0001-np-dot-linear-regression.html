Lesson 0001: What does np.dot do in linear regression?

第 1 课：线性回归里的 np.dot 是什么？

Goal in English: Understand why the teacher's code uses np.dot(X, self.weights) instead of X * self.weights.

中文目标：理解为什么老师代码里用 np.dot(X, self.weights)，而不是 X * self.weights。

1. The beginner story: prediction is weighted addition

English: Linear regression predicts a number by multiplying each feature by its weight, adding the results, and then adding a bias.

中文：线性回归预测一个数：每个特征先乘自己的权重，然后把这些结果加起来，最后再加一个 bias。

prediction = feature_1 * weight_1
           + feature_2 * weight_2
           + feature_3 * weight_3
           + bias

English: If there are many features, writing this by hand is slow and easy to get wrong. np.dot is the NumPy function that does this multiply-then-add pattern for us.

中文：如果特征很多，手写每一项很慢，也容易错。np.dot 就是 NumPy 里帮我们做“对应相乘，然后相加”的函数。

2. Single sample: dot product

English: For one sample, np.dot(x, weights) means: multiply matching positions, then add them.

中文：对一个样本来说，np.dot(x, weights) 的意思是：对应位置相乘，然后把结果加起来。

import numpy as np

x = np.array([1, 2, 3])
weights = np.array([10, 100, 1000])

np.dot(x, weights)
# 1*10 + 2*100 + 3*1000 = 3210

English: The result is one number, because one sample should produce one prediction.

中文：结果是一个数，因为一个样本应该得到一个预测值。

3. Many samples: matrix times vector

English: In the teacher's code, X stores many samples. Each row is one sample, and each column is one feature.

中文：在老师代码中，X 存很多个样本。每一行是一个样本，每一列是一个特征。

ObjectShapeMeaning
X(n_samples, n_features)Many samples, many features.
self.weights(n_features,)One weight for each feature.
np.dot(X, self.weights)(n_samples,)One predicted value for each sample.
X = np.array([
    [1, 2, 3],
    [4, 5, 6],
])
weights = np.array([10, 100, 1000])

np.dot(X, weights)
# array([3210, 6540])

English: The first output, 3210, is the prediction for the first row. The second output, 6540, is the prediction for the second row.

中文：第一个输出 3210 是第一行样本的预测值；第二个输出 6540 是第二行样本的预测值。

4. Why not X * weights?

English: X * weights only multiplies element by element. It does not add the feature contributions together.

中文：X * weights 只是逐元素相乘，它不会把各个特征的贡献加起来。

X * weights
# array([
#   [  10,  200, 3000],
#   [  40,  500, 6000],
# ])

English: This result still has shape (2, 3). It is not the final prediction, because we need one number per sample, not three numbers per sample.

中文：这个结果的 shape 还是 (2, 3)。它不是最终预测，因为我们需要每个样本一个数，而不是每个样本三个数。

5. The teacher's line

English: This is why the correct prediction line in the teacher's implementation is:

中文：所以老师实现里的正确预测代码是：

y_predicted = np.dot(X, self.weights) + self.bias

English: np.dot(X, self.weights) gives one weighted sum per sample. self.bias then shifts every prediction by the same amount.

中文：np.dot(X, self.weights) 给每个样本算出一个加权和。self.bias 再把每个预测值整体平移同样的量。

6. Exam question and answer

Question

English: Why do we use np.dot(X, self.weights) instead of elementwise multiplication X * self.weights for prediction in multiple linear regression?

中文：在多元线性回归中，为什么预测时使用 np.dot(X, self.weights)，而不是逐元素乘法 X * self.weights？

Answer

English: We use np.dot(X, self.weights) because multiple linear regression needs a weighted sum of all features for each sample. The dot product multiplies each feature by its corresponding weight and sums the results, producing one prediction per sample. Elementwise multiplication only multiplies matching entries and does not perform the summation, so it does not directly produce the correct prediction shape.

中文：我们使用 np.dot(X, self.weights)，因为多元线性回归需要对每个样本的所有特征做加权求和。点积会把每个特征乘以对应权重，并把结果加起来，从而为每个样本产生一个预测值。逐元素乘法只会对应位置相乘，不会执行求和，因此不会直接得到正确的预测形状。
