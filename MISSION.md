# Mission: Machine Learning Final Exam Review

## Why
The user is preparing for a closed-book machine learning final exam where questions and answers must be written in English. The teaching goal is to make core course ideas understandable from first principles, then turn them into exam-ready English answers with Chinese translations.

## Success looks like
- Explain each core concept from the required prerequisites without skipping beginner steps.
- Write English-first exam answers for BagOfQuestions-style prompts.
- Recognize and debug common code mistakes in linear regression, logistic regression, neural networks, optimization, and model selection.
- Use teacher-provided files as the trusted source for exam scope and wording.

## Constraints
- `files-from-teacher/` is the highest-trust source.
- `files-from-teacher/` must not be modified unless explicitly requested.
- Questions, exam prompts, answers, key terms, and formula explanations should be English first, then Chinese.
- Chinese explanations are used to build understanding, but the exam-ready form must be English.

## Out of scope
- Non-course machine learning topics unless the teacher materials require them.
- External resources for deciding exam emphasis unless the user explicitly asks.
