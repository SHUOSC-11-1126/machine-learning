Session 1 Linear Regression Cheat Sheet

Session 1 线性回归考试速查表

1. Core Objects

NameShapeEnglish meaning中文
X(n_samples, n_features)Input feature matrix.输入特征矩阵。
y(n_samples,)True target values.真实答案。
weights(n_features,)One weight for each feature.每个特征一个权重。
biasscalarA constant shift added to every prediction.给每个预测加上的常数偏移。
y_predicted(n_samples,)One prediction for each sample.每个样本一个预测值。
dw(n_features,)Gradient for the weights.权重的梯度。
dbscalarGradient for the bias.bias 的梯度。

2. Prediction

English: Linear regression predicts with a weighted sum of features plus a bias.

中文：线性回归用“特征加权求和 + bias”做预测。

y_predicted = np.dot(X, self.weights) + self.bias

3. MSE Loss

English: MSE measures the average squared prediction error.

中文：MSE 衡量预测误差平方的平均值。

math:   L = (1/n) * sum((y_hat - y)^2)
Python: loss = np.mean((y_predicted - y) ** 2)

4. Gradients

English: Gradients tell us how to change parameters to reduce the loss.

中文：梯度告诉我们怎样改变参数才能降低 loss。

dw = (2 / n_samples) * np.dot(X.T, (y_predicted - y))
db = (2 / n_samples) * np.sum(y_predicted - y)

5. Updates

English: Gradient descent subtracts the gradient because the gradient points toward increasing loss.

中文：梯度下降要减去梯度，因为梯度方向是 loss 增大的方向。

self.weights = self.weights - self.lr * dw
self.bias = self.bias - self.lr * db

7. Debug Rules

  • English: Use np.dot(X, self.weights), not X * self.weights, because prediction needs a weighted sum.
  • 中文：预测要用 np.dot(X, self.weights)，不要用 X * self.weights，因为预测需要加权求和。
  • English: Use X.T in dw so the gradient has one value per feature.
  • 中文：dw 里用 X.T，这样梯度才是每个 feature 一个值。
  • English: Update with minus, not plus.
  • 中文：参数更新用减号，不用加号。
