np.dot for Linear Regression

线性回归里的 np.dot 速查表

Core Meaning

English: In linear regression, np.dot(X, weights) computes the weighted sum of features for each sample.

中文：在线性回归中，np.dot(X, weights) 会为每个样本计算“特征乘权重再相加”的结果。

y_predicted = np.dot(X, self.weights) + self.bias

Single Sample

InputMeaningResult
x = [x1, x2, x3]
w = [w1, w2, w3]		 Multiply matching positions, then add them. x1*w1 + x2*w2 + x3*w3

Batch Input

ExpressionShapeMeaning
X(n_samples, n_features)Rows are samples, columns are features.
weights(n_features,)One weight for each feature.
np.dot(X, weights)(n_samples,)One prediction value for each sample.

Do Not Confuse With Elementwise Multiplication

English: X * weights only multiplies matching elements. It does not add across features, so it does not directly produce one prediction per sample.

中文：X * weights 只是逐元素相乘，不会把多个特征加起来，所以它不会直接得到每个样本的一个预测值。

Exam Sentence

English: We use np.dot(X, weights) because multiple linear regression requires a weighted sum over all features for each sample.

中文：我们使用 np.dot(X, weights)，因为多元线性回归需要对每个样本的所有特征做加权求和。
