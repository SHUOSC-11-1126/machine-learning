Code blanks 7–8: update in the negative-gradient direction

The required pattern is parameter = parameter - learning_rate × gradient.

Core knowledge: dw and db are gradients: they describe how the loss changes when the weights or bias changes. To minimize loss, gradient descent subtracts the gradient.

中文：dw、db 是梯度，描述权重或 bias 改变时 loss 怎样变化。为了让 loss 变小，gradient descent 要减去梯度。

Exam-ready English: “Gradient descent updates each parameter by subtracting the learning rate times its gradient, so that the loss decreases.”

中文：梯度下降将每个参数减去学习率乘以其梯度，从而使 loss 降低。
